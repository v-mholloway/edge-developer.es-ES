# notas del producto sobre la privacidad de Microsoft Edge  

La promesa de privacidad de nuestro navegador es ofrecer la protección, transparencia, control y el respeto que merece.  Para mantener los compromisos de ofrecerle transparencia sobre los productos de Microsoft, el equipo de Microsoft Edge proporcionó un documento sobre privacidad que explica cómo funcionan las características y los servicios de Microsoft Edge y cómo cada uno de ellos puede afectar a su privacidad.  El objetivo del equipo de Microsoft Edge es proporcionarle una comprensión completa de cómo se usan sus datos, cómo controlar las distintas características y cómo administrar los datos recopilados, para que tenga la información que necesita para tomar las decisiones de privacidad adecuadas para usted.  

En determinadas secciones del documento, el equipo de Microsoft proporciona pasos para ir a la configuración de Microsoft Edge y a otras páginas.  Por coherencia, el equipo de Microsoft Edge usó un formato abreviado a lo largo de las notas del producto: debería ver direcciones URL que empiezan con `edge://`, como `edge://favorites` o `edge://settings/privacy`.  Para ir a las páginas, escriba el texto en negrita directamente en la barra de direcciones de Microsoft Edge.  Las páginas solo se pueden ver en Microsoft Edge.  

Las notas del producto se centran en la versión de escritorio de Microsoft Edge y algunas partes del documento pueden incluir características o experiencias que no están disponibles para todos los usuarios.  Además, el artículo técnico analiza las características y los servicios que se encuentran en el producto hoy en día, pero pueden estar sujetos a cambios en el futuro.  Microsoft practica la reducción de recopilación de datos, lo que significa que los datos se guardan durante el tiempo mínimo, pero puede variar según la característica o el servicio que se utilice y puede cambiar con el tiempo.  

## Barra de direcciones y sugerencias  

La barra de direcciones le permite especificar direcciones URL de sitio web y buscar en la Web.  De forma predeterminada, la barra de direcciones ofrece búsquedas y sugerencias de sitio con los caracteres que escribe.  Debería ver sugerencias de sus favoritos, el historial de exploración, las búsquedas anteriores y el proveedor de búsquedas predeterminado.  

:::image type="complex" source="./media/whitepaper-media/address-bar.png" alt-text="Barra de direcciones" lightbox="./media/whitepaper-media/address-bar.png":::
   Barra de direcciones  
:::image-end:::  

Para que la navegación y la búsqueda sean más rápidas, a medida que escribe en la barra de direcciones, los caracteres escritos se envían al proveedor de búsqueda predeterminado para que el proveedor de búsqueda envíe consultas de búsqueda sugeridas.  La barra de direcciones categoriza la entrada como una dirección URL, una búsqueda o desconocido.  La información, junto con la sugerencia que seleccione, la posición de la selección y otros datos de la barra de direcciones se enviarán al proveedor de búsqueda predeterminado.  Si su proveedor de búsqueda es Bing, se envía un identificador reconfigurable único para el explorador con los datos para comprender la consulta de búsqueda y la sesión de consulta.  Otros identificadores de servicio de autosugerir se envían al motor de búsqueda predeterminado para completar las sugerencias de búsqueda.  La dirección IP y las cookies se envían al proveedor de búsquedas predeterminado para incrementar la relevancia de los resultados de búsqueda.  Cuando selecciona la barra de direcciones, se envía una señal al proveedor de búsqueda predeterminado para indicar que se prepare para ofrecer sugerencias.  Los caracteres que escriba y las consultas de búsqueda no se enviarán a Microsoft a menos que el proveedor de búsqueda sea Bing.  Para habilitar el envío de datos a su proveedor de búsqueda predeterminado, vaya a `edge://settings/privacy` y, en **Servicios** seleccione **Barra de direcciones** y active la opción **Mostrarme sugerencias de búsqueda y sitios usando los caracteres que escriba**.  Si desactiva la opción, los caracteres escritos ya no se enviarán al proveedor de búsqueda predeterminado.  Las consultas de búsqueda se siguen enviando al proveedor de búsquedas predeterminado para proporcionar resultados de búsqueda.  Si Microsoft Edge detecta que lo escrito en la barra de direcciones puede contener información confidencial, como credenciales de autenticación, nombres de archivo locales o datos de dirección URL que se suelen cifrar, no envía el texto escrito.  Si desea que Microsoft Edge recopile información de diagnóstico sobre la barra de direcciones, incluido el número de consultas ofrecidas para todos los proveedores de búsquedas, vaya a `edge://settings/privacy` y en **Personalice su experiencia web** active la opción **Mejore su experiencia web al permitir que Microsoft use el historial de exploración de esta cuenta para personalizar la publicidad, la búsqueda, las noticias y otros servicios Microsoft**.  

Las pulsaciones de teclas y los sitios web que visite se almacenan de forma local en el dispositivo para cada perfil.  Para eliminar los datos, vaya a `edge://settings/clearBrowserData` y, en la ventana **Borrar datos de exploración**, seleccione la casilla de verificación **Historial de exploración** y seleccione el botón **Borrar ahora**.  Si Bing es su proveedor de búsquedas predeterminado y ha iniciado sesión en Bing, puede eliminar las búsquedas en el [panel de privacidad de Microsoft][MicrosoftAccountPrivacy].  Para borrar el historial de exploración y eliminar los sitios web para que no aparezcan como sugerencias en la barra de direcciones, vaya a `edge://history` y seleccione **Borrar datos de exploración**.  Para borrar los datos que Microsoft recopila de la barra de direcciones y las sugerencias de búsqueda en Windows 10, vaya a **Inicio** > **Configuración** > **Privacidad** > **Comentarios y diagnósticos** y, en **Eliminar datos de diagnóstico**, seleccione **Eliminar**.  El resto de los datos se eliminarán después de 36 meses.  

Si ha iniciado sesión en Microsoft Edge con una cuenta profesional o educativa de Microsoft y Microsoft Search está disponible, se enviará un token anónimo que representa la cuenta con la consulta para ofrecer la funcionalidad específica de la cuenta, como resultados específicos para su empresa.  

Todos los datos se transmiten de forma segura a través de HTTPS.  Si [Bing][|::ref1::|Main] es su proveedor de búsqueda predeterminado, las búsquedas y los caracteres escritos se guardan hasta 6 meses.  

Si busca una sola palabra en el cuadro de dirección, es posible que Microsoft Edge envíe la misma palabra al servidor DNS para ver si corresponde a un host de la red y puede intentar conectar con el host correspondiente.  La opción le permite desplazarse hasta el host específico en lugar de realizar la búsqueda.  Por ejemplo, si su enrutador usa el nombre de host `router` y escribe `router` en la barra de direcciones, se le proporciona la opción de ir a `https://router`, así como de buscar la palabra `router` con el proveedor de búsqueda predeterminado.  La característica no está controlada por la opción **Mostrarme las sugerencias de búsqueda y sitio con los caracteres escritos** ya que no implica enviar datos al motor de búsqueda predeterminado.  

Otro método para habilitar \(o deshabilitar\) el envío de datos a su proveedor de búsquedas predeterminado es ir a `edge://settings/search` y activar \(o desactivar\) la opción **Mostrar sugerencias de búsqueda y sitio con los caracteres escritos**.  Para cambiar el motor de búsqueda predeterminado, vaya a `edge://settings/search` y seleccione el menú desplegable **Motor de búsqueda usado en barra de direcciones**.  Si está explorando mientras usa el modo de InPrivate o de invitado, las seugerencias automáticas se desactivan.  InPrivate muestra sugerencias de la navegación local, como el historial de exploración o las búsquedas pasadas, pero no se envían caracteres escritos al motor de búsqueda predeterminado.  El modo de invitado no muestra ninguna sugerencia ni envía caracteres escritos al motor de búsqueda predeterminado.  

Los datos recopilados por otros proveedores de búsqueda siguen la directiva de privacidad de la empresa.  

## Autorrellenar  

Autorrellenar en Microsoft Edge le ayuda a ser más productivo ya que permite guardar contraseñas, información de pago, direcciones y otros datos de entrada de formularios.  Cuando visita un sitio y empieza a rellenar un formulario, Microsoft Edge usa la información de relleno de formularios para coincidir con los datos de autorrellenar guardados en el formulario.  Microsoft Edge ofrece los datos de entrada de formulario que guardó anteriormente cuando encuentra formularios similares.  Las contraseñas y la información de la tarjeta de crédito solo se guardan con el permiso explícito para cada contraseña y tarjeta.  

Las direcciones y otras entradas de formularios se guardan de forma predeterminada.  Para deshabilitar la opción de guardar y autorrellenar de la dirección y otros datos de formulario, vaya a `edge://settings/addresses` y desactive la opción **Guardar y rellenar direcciones**.  

Para impedir que Microsoft Edge le pida que guarde las contraseñas, vaya a `edge://settings/passwords` y desactive la opción **Ofrecer guardar contraseñas**.  Si no quiere que Microsoft Edge rellene automáticamente las contraseñas guardadas existentes y desea eliminar las contraseñas guardadas, vaya a `edge://settings/passwords` y seleccione **Contraseñas guardadas**.  Para eliminar todos los datos de Autorrellenar, vaya a `edge://settings/clearBrowserData`, seleccione **Autorrellenar datos de formulario**, seleccione el intervalo de tiempo que prefiera y, después, haga clic en **Borrar ahora**.  

Si ha activado la sincronización para su perfil, los datos de Autorrellenar se sincronizan en todas las versiones de Microsoft Edge en que ha iniciado sesión con las mismas credenciales.  Cuando la sincronización está activada, todos los datos de Autorrellenar se almacenan en servidores de Microsoft cifrados.  Los datos de Autorrellenar guardados en servidores de Microsoft se usan solo para propósitos de sincronización.  Para desactivar la sincronización de los datos de Autorrellenar, vaya a `edge://settings/profiles/sync` y al lado su Perfil seleccione el botón **Desactivar la sincronización**.  Si activó la sincronización de Autorrellenar, al eliminar datos de Autorrellenar de un dispositivo mientras haya iniciado sesión en Microsoft Edge, se quitarán los datos de Autorrellenar de todos los demás dispositivos que hayan iniciado sesión en la misma cuenta.  

Cuando visita una página web y envía un formulario, Microsoft Edge envía información sobre el formulario, como un hash del nombre de host y los tipos de entrada de Autorrellenar \(por ejemplo, cuadro 1 está buscando una dirección de correo electrónico, cuadro 2 está buscando una contraseña y así sucesivamente), al servicio de relleno de formularios de Microsoft.  Ninguna información especificada por el usuario o identificadores de usuario se envían al servicio.  La información ayuda a Microsoft Edge a identificar correctamente formularios en distintas páginas web.  Los datos se usan para ayudar a asignar los datos de Autorrellenar guardados en el formulario.  

Cuando se usa el modo de invitado, Autorrellenar no está disponible y no se agregan nuevas entradas de Autorrellenar.  Al navegar en InPrivate, Microsoft Edge ofrece entradas de Autorrellenar, pero no se agregan nuevas entradas de Autorrellenar.  

## Cast  

Cast en Microsoft Edge, le permite mostrar los elementos multimedia en otra pantalla con Google Cast.  Para obtener acceso a la característica Cast, abra **Configuración y más (...)** > **Más herramientas** y seleccione **Cast multimedia al dispositivo**.  Cast se basa en la extensión Media Router que no se incluye en Microsoft Edge de manera predeterminada.  La primera vez que usa Cast, Microsoft Edge pide permiso para instalar la extensión Media Router.  

Seleccione **reiniciar** para instalar las extensiones de Media Router desde Chrome Web Store.  Para mantener la extensión Media Router actualizada, al iniciar Microsoft Edge y de forma periódica, Microsoft Edge envía solicitudes de actualización a Chrome Web Store que contienen datos básicos sobre la versión de Microsoft Edge.  Es posible que Google recopile datos asociados con la extensión Media Router.  Para desinstalar la extensión Media Router, vaya a `edge://flags` y deshabilite **Edge-On-Demand-Media-Router**.  Esto también detiene las actualizaciones desde Chrome Web Store.  La extensión está oculta y no aparece en la lista **Extensiones instaladas**.  En la lista **Extensiones instaladas**, vaya a `edge://extensions`.  

## Colecciones  

Puede recopilar sitios, texto e imágenes en la Web y organizar el contenido con Colecciones en Microsoft Edge.  Todos los datos de colecciones se almacenan de forma local en el dispositivo y se organizan según el perfil de Microsoft Edge.  Si tiene la sincronización activada para Colecciones, las colecciones creadas, incluidas las notas o comentarios, están disponibles en todas las versiones de Microsoft Edge con la sesión iniciada y sincronizadas.  

Cada 24 horas, Microsoft Edge descarga una lista de los sitios admitidos para los que existe una plantilla de extracción de entidad especial.  Las plantillas son específicas de cada sitio web.  Cuando se crea un elemento nuevo en la colección, Microsoft Edge comprueba que el sitio desde el que se está creando el nuevo elemento de colección se encuentra en la lista de sitios admitidos.  Si el sitio se encuentra en la lista, Microsoft Edge hace ping al servicio de extracción de entidades para la plantilla del sitio específico.  No hay ningún identificador de usuario asociado a la solicitud al servicio.  La plantilla intenta identificar el nombre, el precio, las calificaciones, la imagen principal y otros datos sobre el elemento que se va a recopilar.  Si el sitio desde el que va a crear un nuevo elemento de colección no se encuentra en el sitio de lista admitido, Microsoft Edge no descarga una plantilla.  Las plantillas permiten la creación de todos los elementos de colección de forma local en el dispositivo.  No se envían datos sobre los elementos de colección al servicio para crear la colección.  

Para eliminar las plantillas almacenadas en el dispositivo y borrar los datos de la caché, vaya a `edge://settings/privacy` y, en **Borrar datos de exploración** junto a **Borrar los datos de exploración ahora** seleccione el botón **Elegir lo que se debe borrar**, seleccione el intervalo de tiempo y el tipo de datos que quiera y, después, seleccione el botón **Borrar ahora**.  Otro método para eliminar datos almacenados en caché es ir a `edge://settings/clearBrowserData`, seleccionar el intervalo de tiempo y el tipo de datos que desee y seleccionar el botón **Borrar ahora**.  

Si activa la opción **Mostrar sugerencias de Pinterest en Colecciones**, Colecciones realiza una búsqueda de Microsoft Bing con el título de la colección para encontrar páginas de tema relevantes de Pinterest.  Microsoft Edge no envía datos sobre sus colecciones a Pinterest.  Para quitar las sugerencias y detener las búsquedas de las páginas de tema de Pinterest, vaya a `edge://settings/privacy` y desactive la opción **Mostrar sugerencias de Pinterest en Colecciones**.  

Colecciones no está disponible cuando se usa la exploración InPrivate o en modo de invitado.  

## Bloqueos  

Si los datos opcionales de diagnóstico, incluidos los informes de bloqueo, están activados, Microsoft recopila datos de diagnóstico cuando Microsoft Edge se bloquea o detecta otros problemas de confiabilidad.  Los datos de diagnóstico se usan para diagnosticar y solucionar problemas de confiabilidad de Microsoft Edge y otros productos y servicios de Microsoft.  

:::image type="complex" source="./media/whitepaper-media/crashes2.png" alt-text="Bloqueos" lightbox="./media/whitepaper-media/crashes2.png":::
   Bloqueos  
:::image-end:::  

La información de diagnóstico recopilada se encuentra en forma de volcados de memoria, que contiene el estado de dispositivo y software capturado en el momento en que Microsoft Edge encontró el problema de confiabilidad.  El volcado de memoria contiene información sobre lo que se estaba realizando en el momento del problema de confiabilidad.  Se puede incluir información como el sitio web que visitaba en el momento del bloqueo o el uso de la CPU en los datos de diagnóstico.  Los datos de diagnóstico de bloqueo se almacenan de forma local en el dispositivo y se envían a Microsoft a través de un vínculo cifrado cuando el informe de bloqueo está activado.  Cada volcado de memoria contiene un identificador único del dispositivo, un identificador configurable único para el explorador, y otros datos de diagnóstico (como la URL, el uso de la CPU y el uso de la red) para ayudar a identificar el problema.  Los datos de diagnóstico adicionales se adjuntan al archivo de volcado de memoria para ayudar a diagnosticar el problema de confiabilidad, como, por ejemplo, entender cuántos dispositivos están experimentando el problema y la gravedad.  

Los volcados de memoria se envían a Microsoft y se almacenan en servidores de Microsoft seguros durante 30 días y, después, se eliminan.  Para solicitar la eliminación de los datos de diagnóstico en dispositivos con Windows 10, vaya a**Inicio** > **Configuración** > **Privacidad** > **Comentarios y diagnósticos** y en **Eliminar datos de diagnóstico** seleccione **Eliminar**.  La información de bloqueo agregada, como un recuento de los tipos de bloqueos que se producen, se almacena para la creación de informes y propósitos de mejora del producto.  

Para borrar los datos de diagnóstico de bloqueo almacenados de forma local desde el sistema de archivos del dispositivo, vaya a `edge://crashes` y seleccione el botón **Borrar todo**.  

Para desactivar la recopilación de datos de diagnóstico de bloqueo en Windows 10, vaya a**Inicio** > **Configuración** > **Privacidad** y seleccione **Comentarios y diagnósticos**.  Para las versiones de Microsoft Edge en todas las demás plataformas, vaya a `edge://settings/privacy` y desactive la opción **Mejorar los productos de Microsoft enviando informes de bloqueo y datos sobre cómo usa el explorador**.  También puede desactivar la característica de recopilación de datos de diagnóstico para las empresas mediante [directivas de grupo administradas por la organización][DeployedgeEnterprisePrivacySettings].  

## Herramientas de desarrollo

Las herramientas de desarrollo de Microsoft Edge proporcionan herramientas para depurar y probar sitios Web. Para acceder a las herramientas de desarrollo, abra **Configuraciones y más (...)** > **Más herramientas** y seleccione **Herramientas para desarrolladores**.  Al activar algunas características de las herramientas de desarrollo, Microsoft Edge pide módulos de los servidores de Microsoft y los descarga en el dispositivo.  La solicitud de los módulos enviados se encuentra en una conexión HTTPS segura y contiene un identificador no único que representa la versión de Microsoft Edge que se usa.  Las experiencias específicas que requieren la descarga remota incluyen la vista 3D y el panel de accesibilidad de la herramienta elementos.  La integración de Webhint requiere un módulo remoto que se pide automáticamente cuando se abre la Herramienta de desarrolladores.  

## Datos de diagnóstico sobre cómo usar el explorador  

Microsoft usa los datos de diagnóstico para mejorar los productos y servicios de Microsoft, mantener los productos de Microsoft seguros y actualizados, y ayudar a comprender mejor el rendimiento de los productos de Microsoft. Siempre que el equipo de Microsoft Edge recopile datos, la decisión se verificará como su elección adecuada.  Microsoft cree en la reducción de la recopilación de información y la pone en práctica.  El equipo de Microsoft Edge se esfuerza por recopilar únicamente la información necesaria y la almacena solo durante el tiempo que sea necesario para mejorar los productos y servicios de Microsoft.  

A medida que use las características y los servicios de Microsoft Edge y otras aplicaciones que usan la plataforma web de Microsoft Edge, los datos de diagnóstico sobre cómo usa estas características se enviarán a Microsoft.  Los datos de diagnóstico incluyen información como la instalación de Microsoft Edge, el uso de características, el rendimiento y el uso de memoria.  Por ejemplo, si marca un sitio web como favorito, el equipo de Microsoft Edge recibe la información de que se hizo clic en el botón de favorito y se agregó un favorito correctamente, pero no el sitio que se ha marcado como favorito.  Los datos de diagnóstico también incluyen información de rendimiento, como cuántos milisegundos tardó en abrir una nueva pestaña.  Las características y los servicios que se mencionan en el documento recopilan datos de diagnóstico.

   Datos de diagnóstico  
:::image-end:::  

Además, Microsoft Edge recopila un conjunto de datos de diagnóstico requeridos que son necesarios para mantener el producto actualizado, seguro y funcionando correctamente.  Entre los datos de diagnóstico necesarios se incluye la conectividad del dispositivo, la información de configuración sobre las opciones del consentimiento de la recopilación de datos actual, la versión de la aplicación y el estado de instalación.  Solo puede desactivar la configuración con directivas de grupo que administre su organización.  Para más información sobre los datos de diagnóstico en su organización, consulte [Configurar datos de diagnóstico de Windows en su organización][WindowsPrivacyConfigureDiagnosticDataOrganization].  

Microsoft Edge genera los datos de diagnóstico, los almacena de forma local y los envía periódicamente a Microsoft.  Los datos de diagnóstico se envían mediante HTTPS y se almacenan en servidores de Microsoft.  Los datos de diagnóstico se asocian a un identificador único al dispositivo y un identificador configurable único para el explorador.  Los identificadores no contienen información personal.  Para restablecer el identificador único para el explorador en dispositivos con Windows 10, vaya a **Inicio** > **Configuración** > **Privacidad** > **Comentarios y diagnósticos** y, en **Eliminar datos de diagnóstico**, seleccione **Eliminar** o cambie la opción en **Datos de diagnóstico** de **Completo** a **Básico**.  En otras plataformas, para generar un nuevo identificador \(ID\) reconfigurable que sea único para el explorador, vaya a `edge://settings/privacy` y desactive la opción **Mejorar los productos de Microsoft enviando datos sobre cómo usa el explorador**.  Es posible que la funcionalidad de restablecimiento del ID. sea diferente en los dispositivos administrados con directivas de grupo establecidas por la organización.  

Para ver los datos de los productos que se han compartido con Microsoft en el visor de datos de diagnóstico de Windows 10, versión 1803 \ (2018 de abril \) o más recientes, abra **Inicio** > **Configuración** > **Privacidad** > **Comentarios y diagnóstico** y en **Ver datos de diagnóstico** seleccione **Abrir Visor de datos de diagnóstico**.  

Para otras plataformas o versiones de la versión 1803 de Windows 10 y versiones anteriores, para ver los datos de diagnóstico, vaya a `edge://data-viewer`.  Para ver los datos que se envían periódicamente a Microsoft desde la última vez que se abrió el visor, vaya a `edge://data-viewer`.  Para ver los datos que se han enviado a Microsoft para su sesión específica, actualice el visor.  Los datos que se usan para rellenar `edge://data-viewer` se almacenan de forma local en el dispositivo.  Para borrar los datos en el visor, cierre la pestaña `edge://data-viewer`.  

Los datos de diagnóstico se almacenan en servidores de Microsoft hasta 18 meses.  Para eliminar los datos de diagnóstico en Windows 10, vaya a **Inicio** > **Configuración** > **Privacidad** > **Comentarios y diagnósticos** y, en **Eliminar datos de diagnóstico**, seleccione **Eliminar**.  La funcionalidad Eliminar datos de diagnóstico solo se admite en la versión 1803 o posterior de Windows 10.  Para más información, consulte [Diagnósticos, comentarios y privacidad en Windows 10][MicrosoftSupport4468236].  

En el caso de Microsoft Edge para Windows 10, la opción **Mejorar los productos de Microsoft enviando datos sobre cómo usa el explorador** viene determinada por la configuración de datos de diagnóstico de Windows.  Para ver el estado de la configuración, vaya a `edge://settings/privacy`.  Para cambiar la configuración de Windows, abra **Inicio** > **Configuración** > **Privacidad** y seleccione **Comentarios y diagnósticos**.  En todas las demás plataformas, para controlar la recopilación de datos de diagnóstico, vaya a `edge://settings/privacy` y active o desactive la opción **Mejorar los productos de Microsoft enviando informes de bloqueo y datos sobre cómo usa el explorador**.  Esta configuración se usa para todos los perfiles asociados a la instalación de Microsoft Edge en un dispositivo.  La configuración no está sincronizada entre dispositivos.  La configuración se aplica a la navegación InPrivate y al modo de invitado.  Si el dispositivo se administra con directivas de grupo establecidas por su organización, vaya a `edge://settings/privacy`.  

## Administración de derechos digitales y licencias multimedia  

Cuando un sitio web ofrece contenido multimedia que está protegido por la administración de derechos digitales \(DRM\), Microsoft Edge usa una canalización de reproducción segura para asegurarse de que el contenido no se copia ni se obtiene acceso a él de forma incorrecta.  Como parte de la característica, Microsoft Edge puede almacenar datos relacionados con DRM en el dispositivo, incluido un identificador único y licencias de medios, y puede transmitir el identificador único a un servidor de licencias multimedia especificado por el proveedor del contenido.  Cuando usa el sitio web, Microsoft Edge recupera la información DRM para asegurarse de que tiene permiso para usar el contenido.  Los datos ayudan a validar el acceso al contenido protegido y a garantizar una experiencia multimedia perfecta.  

Microsoft Edge es compatible con DRM mediante la API de Extensiones multimedia cifradas \(EME API\) para sitios de HTML5.  La API EME permite a los sitios web comunicarse con un proveedor DRM denominado Módulo de descifrado de contenido \(CDM\).  Es posible que los sistemas DRM distintos, como Widevine por Google o PlayReady por Microsoft, puedan ser compatibles con la implementación CDM del desarrollador.  Los proveedores de contenido pueden optar por admitir uno o más de los posibles sistemas DRM y pueden usar la funcionalidad de la API EME para decidir qué sistema DRM usar para un cliente específico.  Para obtener más información acerca de la privacidad de EME, vea [Privacidad de extensiones multimedia cifradas][W3cEncryptedMediaPrivacy].  

Microsoft Edge admite DRM de PlayReady solo en Windows 10.  PlayReady es una implementación DRM para ofrecer experiencias multimedia como vídeo 4K y audio Dolby Atmos.  Microsoft Edge usa las API de Windows Platform Media Foundation para admitir PlayReady.  Para validar el acceso al contenido protegido, Microsoft Edge usa el sistema operativo Windows 10, que usa un identificador único \(ID.\) y comunica el identificador con el servicio PlayReady.  Todos los datos de los EME, CDM y el explorador de PlayReady que continúan en el dispositivo se almacenan y mantienen en Microsoft Edge.  Para más información sobre PlayReady, consulte [Sistema de un extremo a otro extremo sencillo][PlayreadyOverviewSimpleEndSystem].  

Microsoft Edge admite Widevine por DRM de Google y la opción está activada de forma predeterminada.  Microsoft Edge captura actualizaciones periódicamente para Widevine de los servidores de Google.  El uso de Widevine puede incluir las comunicaciones con Google.  Para optar por no usar Widevine en Microsoft Edge, vaya a `edge://flags/#edge-widevine-drm` y desactive la opción Widevine DRM.  Widevine tiene la capacidad de crear un identificador de dispositivo único y transmitirlo a Google.  Para obtener información más específica sobre Widevine y la privacidad, vea la Directiva de privacidad de Google.  

Microsoft Edge es compatible con la DRM de Flash Access de Adobe, lo que usan algunos sitios en lugar de HTML5.  Debe conceder permiso para permitir Adobe Flash cuando un sitio lo solicite.  Cuando un sitio usa la DRM de Flash Access de Adobe, Microsoft Edge concede a Adobe acceso a un identificador de dispositivo único.  Para desactivar y restablecer las instancias almacenadas de forma local, vaya a `edge://settings/privacy` y, en **Borrar datos de exploración** seleccione **Elegir lo que se debe borrar**, marque la casilla de verificación **Cookies y otros datos del sitio** y seleccione **Borrar ahora** para quitar todos los identificadores almacenados.  Para impedir el uso de Adobe Flash DRM, vaya a `edge://settings/content/flash`.  

Cuando solicita acceso al medios HTML5 codificados como una película en línea, Microsoft Edge crea una solicitud de licencia para descifrar los archivos multimedia.  El CDM que se usa crea la solicitud de licencia que contiene un identificador de solicitud.  La solicitud se envía al servidor de licencias.  Ninguna parte de la solicitud de licencia contiene información de identificación personal y la solicitud de licencia no se almacena en el dispositivo.  

Cuando devuelve la licencia multimedia, se crea un identificador del contenido multimedia que es único para el usuario y el sitio.  El Id. no se comparte entre sitios y es diferente para cada sitio.  Un identificador de sesión, que se usa para identificar una sesión de reproducción, se envía con el identificador del contenido multimedia para descifrar el contenido multimedia.  El identificador del contenido multimedia se almacena de forma local en el dispositivo y puede almacenarse con el proveedor de contenido.  

Para desactivar todas las protecciones DRM y de contenido, vaya a `edge://settings/content/protectedContent` y desactive las opciones **Permitir que los sitios reproduzcan contenido protegido (recomendado)** y **Permitir el uso de identificadores para contenido protegido (es posible que se requiera reiniciar el equipo)**.  

*   La desactivación de la opción **Permitir que los sitios puedan reproducir contenido protegido** deshabilita la reproducción de los sistemas DRM basados en CDM, como PlayReady y Widevine, pero no en los sistemas no basados en CDM, como DRM de Flash Access.  Para administrar los permisos del sitio de Flash, vaya a `edge://settings/content/flash`.  Si desactiva la configuración, la funcionalidad multimedia dejará de funcionar correctamente.  
*   Desactivar la opción **Permitir identificadores de contenido protegido** impide la creación de identificadores para la DRM de Flash Access y evita que Widevine recupere periódicamente las actualizaciones de Google.  Si la configuración está desactivada, es posible que la funcionalidad multimedia de algunos sitios deje de funcionar correctamente.  

## No realizar seguimiento  

Para habilitar **No realizar seguimiento** en Microsoft Edge, vaya a `edge://settings/privacy` y active la opción **Enviar solicitudes de "No realizar seguimiento"**.  Si habilita la característica **No realizar seguimiento**, Microsoft Edge envía un encabezado DNT:1 HTTP con solicitudes de tráfico saliente de HTTP, HTTPS y SPDY a los sitios web que visite para solicitar a cada uno no use rastreadores.  Sin embargo, habilitar la opción **Enviar solicitudes de "No realizar seguimiento"** no garantiza que los sitios web no puedan rastrearlo.  Algunos sitios pueden respetar su solicitud mostrando anuncios que no se basan en la exploración anterior.  Microsoft Edge no controla si se accede a la solicitud.  Para ayudar a evitar que los sitios web realicen un seguimiento de su usuario, vaya a `edge://settings/privacy` y cambie la configuración de **Prevención de seguimiento** a **Equilibrada** o **Estricta**.  

Cuando se usa el modo de invitado, Microsoft Edge no envía solicitudes de **No realizar seguimiento**.  Cuando se usa la exploración InPrivate, Microsoft Edge solo envía solicitudes de **No realizar seguimiento** si la opción **Enviar solicitudes de "No realizar seguimiento"** está activada para el perfil que está usando.  

## Descargas  

Microsoft Edge le permite descargar archivos de forma segura.  Puede elegir dónde se descargarán los archivos en el dispositivo en `edge://settings/downloads`.  Si SmartScreen está habilitado, se enviará información sobre el archivo, como el nombre de archivo y la dirección URL, a SmartScreen para comprobar la reputación del archivo.  La comprobación de la reputación le ayuda a evitar descargar accidentalmente malware conocido que se sabe que perjudicará al dispositivo.  Para activar el filtro SmartScreen \ (o desactivado \), vaya a `edge://settings/privacy` y active el filtro SmartScreen.  Para más información sobre SmartScreen, vea la sección [SmartScreen](#smartscreen).  

Para ver el historial de las descargas anteriores, vaya a `edge://downloads`.  Para borrar los datos de exploración y eliminar el historial de descargas, vaya a `edge://settings/clearBrowserData`.  Si elimina el historial de descargas de Microsoft Edge, no se eliminarán los archivos del dispositivo.  La eliminación de los archivos descargados desde el dispositivo no quita los archivos del historial de descargas.  Cuando se usa el modo de exploración de InPrivate o de invitado, el historial de descargas de la sesión se elimina cuando se cierran las ventanas de invitado o InPrivate, pero los archivos se guardan en el dispositivo.  

## Extensiones y complementos de Microsoft Edge  

Puede instalar extensiones en Microsoft Edge para agregar funciones al explorador.  Cuando instala una extensión desde el sitio web de complementos de Microsoft Edge o desde otro almacén de extensiones, Microsoft recopila información sobre la extensión para ayudar a los desarrolladores y a Microsoft a comprender cómo se usa la extensión.  Microsoft Edge recopila datos agregados, incluido el número de veces que se ha descargado una extensión y la información sobre el rendimiento, como los datos de bloqueo.  Microsoft comparte los datos agregados con los desarrolladores de la extensión.  Los comentarios y las opiniones de los usuarios son públicos en el sitio web de complementos y también se comparten con los desarrolladores. Si ha iniciado sesión en Microsoft Edge, las extensiones instaladas desde el sitio web de complementos de Microsoft Edge se asocian con su cuenta para ofrecer recomendaciones de extensión.  Los datos se usan en conjunto para comprender la popularidad de las extensiones.   

Si desea que sus extensiones y preferencias de extensiones se sincronicen en todas las versiones en sincronización de Microsoft Edge con la sesión iniciada, vaya a `edge://settings/profiles/sync` y seleccione el botón **Activar sincronización**.   

La instalación de extensiones es opcional, para desinstalar cualquier extensión en cualquier momento, vaya a `edge://extensions`.  Cuando se instala una extensión, especifica los datos de usuario a los que necesita tener acceso y Microsoft Edge pide permiso antes de instalar la extensión.  Asegúrese siempre de que una extensión es creíble y segura antes de instalarla y revise la directiva de privacidad del desarrollador para la extensión específica.  

Las extensiones se actualizan con el servicio de actualización de Microsoft Edge.  Microsoft Edge envía una lista de extensiones instaladas al servicio de actualización para comprobar si hay una actualización disponible.  Si instala una extensión desde Chrome Web Store, las solicitudes se enviarán a Chrome Web Store periódicamente para comprobar si hay actualizaciones de extensión.  El identificador de extensión, la versión de extensión y la información sobre Microsoft Edge se incluyen en la solicitud para buscar actualizaciones.  Para detener las solicitudes de Chrome Web Store, vaya a `edge://extensions` y en **De otros orígenes**, desinstale las extensiones.  

Cuando se importan extensiones de otros exploradores como Google Chrome, si una extensión está disponible en el sitio web de complementos de Microsoft Edge, Microsoft Edge instala automáticamente la extensión desde el sitio web de complementos de Microsoft Edge y la activa si previamente tenía la extensión activada.  Si la extensión no se encuentra disponible en el sitio web de complementos de Microsoft Edge, Microsoft Edge copia de forma local e instala su extensión desde Google Chrome sin tener que activar o conectar con Chrome Web Store.  Microsoft Edge pide permiso para activar la extensión y permitir extensiones de otras tiendas.  Si ha concedido permiso, Microsoft Edge permite la instalación de extensiones de otras tiendas y actualizaciones de las extensiones mediante Chrome Web Store.  Para habilitar \(o deshabilitar\) la opción, vaya a `edge://extensions` y active \(o desactive\) **Permitir extensiones de otras tiendas**.

## Seguridad familiar  

Microsoft ofrece varias herramientas para ayudar a las familias a mantenerse conectadas y mejorar la seguridad de los niños en dispositivos con Windows 10, Xbox y Android que ejecuten Microsoft Launcher.  

En un grupo de familia, hay opciones de configuración de familia que deben habilitarse para niños mientras se usa Microsoft Edge.  El organizador del grupo de familia debe habilitar las opciones para los usuarios del grupo.  Las tres características principales ofrecidas a un grupo de familia son el filtrado web, el informe de actividades y la búsqueda segura.  

El filtrado web evita que los usuarios infantiles de un grupo familiar vayan a sitios web de contenido adulto o sitios web que estén específicamente bloqueados por el organizador familiar.  

El informe de actividad registra información acerca de los sitios web que los niños visitan, así como las búsquedas, el tiempo de pantalla, los dispositivos utilizados y los intentos de visitar sitios bloqueados.  El organizador de grupos de familia puede ver la información en [family.microsoft.com][MicrosoftAccountFamilyMain].  Los datos se recopilan, se cifran en tránsito, se envían a Microsoft y se almacenan en servidores de almacenamiento seguros de Microsoft.  Los datos se recopilan con la cuenta de Microsoft del niño para que puedan administrarse correctamente.  Los informes de actividades se almacenan en [family.microsoft.com][MicrosoftAccountFamilyMain] durante un máximo de 30 días y, después, se eliminan.  

La búsqueda segura agrega una palabra clave segura a la solicitud de encabezado para los motores de búsqueda.  Bing lee la palabra clave segura y filtra los resultados de búsqueda que se devuelven al niño.  Otros motores de búsqueda también pueden devolver resultados filtrados por la palabra clave.  Todas las búsquedas de los niños se recopilan y se ponen a disposición del organizador familiar para que las vea en los informes de actividades o en [family.microsoft.com][MicrosoftAccountFamilyMain].  Los datos se recopilan con la cuenta de Microsoft del niño para que puedan administrarse correctamente.  

Los datos de exploración del niño se almacenan en servidores de Microsoft seguros y se ponen a disposición de los padres durante un máximo de 30 días y, después, se eliminan de inmediato.  Los datos pueden eliminarse en cualquier momento desde el [panel de privacidad de Microsoft][MicrosoftAccountPrivacy].  Para borrar los datos de exploración almacenados de forma local en un dispositivo, vaya a `edge://settings/clearBrowserData`.  

El niño debe haber iniciado sesión en Windows 10 con una cuenta de Microsoft, y la opción de informes de actividades debe activarla el organizador familiar para permitir la recopilación de los datos y el uso compartido con el organizador del grupo familiar.  No es necesario que el niño haya iniciado sesión en Microsoft Edge para recopilar los datos.  Si las características de seguridad familiar no están disponibles en su versión de Windows 10, puede actualizar a la versión más reciente de Windows para obtener las características.  

El modo de invitado y la exploración de InPrivate se deshabilitan si está activado el filtro web o el informe de actividades.  

El organizador del grupo familiar puede detener la recopilación de datos desde el portal de seguridad familiar.  Para más información sobre las características de seguridad de familiar de Microsoft, consulte [¿Qué es un grupo de familia de Microsoft?][MicrosoftSupport12413]  

## Geolocalización  

Microsoft Edge es compatible con la [API de ubicación geográfica][W3cGeolocationApiMain], que permite a los sitios web tener acceso a la información de la ubicación con su permiso.  Los sitios web pueden pedirle la ubicación, por ejemplo, al intentar encontrar el cibercafé más cercano.  Microsoft Edge siempre pide permiso para conceder a los sitios web acceso a la ubicación.  Para administrar el permiso o para bloquear siempre el acceso a la ubicación de los sitios, vaya a `edge://settings/content/location`.  

En el lado derecho de la barra de direcciones, Microsoft Edge indica cuándo se comparte o no su ubicación.  

:::image type="complex" source="./media/whitepaper-media/geolocation2.png" alt-text="Ubicación" lightbox="./media/whitepaper-media/geolocation2.png":::
   Ubicación  
:::image-end:::  

Si permite el uso compartido de su ubicación con un sitio, Microsoft Edge envía información de red local, como su dirección IP y los puntos de acceso Wi-Fi cercanos, al servicio de ubicación de Microsoft.  El servicio de Microsoft usa la información para calcular las coordenadas de ubicación geográfica.  La estimación de ubicación geográfica se comparte con el sitio con el que acordó compartir su ubicación.  Para especificar que Microsoft Edge proporciona al sitio que lo solicita una ubicación más precisa en Windows 10, vaya a **Configuración** > **Privacidad** > **Ubicación** y active las opciones **Permitir el acceso a la ubicación de este dispositivo** y **Permitir que las aplicaciones accedan a tu ubicación**.  Si desactiva las opciones **Permitir el acceso a la ubicación en este dispositivo** y **Permitir que las aplicaciones obtengan acceso a su ubicación** la configuración, Microsoft Edge ofrece una ubicación aproximada al sitio que lo solicite.  La información solo se comparte con un sitio que la solicite si previamente permitió ver la ubicación.  Para más información sobre la configuración de la ubicación de Windows, consulte [Servicio de ubicación de Windows 10 y privacidad][MicrosoftSupport4468240].  

Se usa un nuevo identificador generado de forma aleatoria al realizar solicitudes al servicio de ubicación de Microsoft Edge.  El servicio de ubicación de Microsoft Edge no almacena las coordenadas de ubicación geográfica durante ningún período de tiempo.  

La exploración de InPrivate usa la configuración de permisos de ubicación del perfil desde el que se inició la sesión de InPrivate.  El modo de invitado siempre pide permiso antes de proporcionar la ubicación al sitio.  

## Importar datos de explorador  

Microsoft Edge ofrece una experiencia interactiva y fluida cuando inicia el navegador por primera vez.  Como parte de la experiencia, tiene la opción de importar los datos del explorador a Microsoft Edge desde otro explorador.  Durante la experiencia de importación, puede mantener los datos importados, o eliminarlos y empezar de cero.  Los datos incluyen sus favoritos, el historial de exploración, los datos de Autorrellenar, las extensiones, la configuración y otros datos de exploración.  Los datos de exploración de versiones anteriores de Microsoft Edge se importan automáticamente cuando actualiza al nuevo Microsoft Edge.  Con su confirmación, Microsoft Edge importa datos de otros exploradores, como Google Chrome, Mozilla Firefox o Internet Explorer, y el navegador desde el que se importan depende del navegador más usado, según se haya definido en el sistema operativo.  La importación de los datos se completa de forma local en el dispositivo, se almacena de forma local y no se envía a Microsoft, a menos que acepte sincronizar los datos de exploración.

Cuando se importan extensiones desde un explorador diferente, como Google Chrome, Microsoft Edge importa una copia local y pide permiso antes de que se active en caso de que la extensión no esté disponible en el sitio web de complementos de Microsoft Edge.  Es posible que hayan cambiado los permisos de algunas de las extensiones.  Para revisar los permisos de extensión, vaya a `edge://extensions`.  

Puede importar datos desde otro navegador en cualquier momento en `edge://settings/importData`.  

## Instalar y actualizar  

Puede descargar e instalar Microsoft Edge en plataformas como Windows y macOS.  Microsoft Edge usa el servicio de actualización para mantener la versión de Microsoft Edge actualizada y protegida.  

Al descargar e instalar Microsoft Edge, la información sobre el dispositivo, como el canal de lanzamiento, información básica de hardware, identificadores de actualización, un identificador único para el dispositivo y un identificador configurable único para su explorador se enviarán a Microsoft durante el proceso de instalación.  La dirección IP del dispositivo se envía al servicio de actualización, pero el último decimal se borra para las protecciones de privacidad agregadas.  Durante cada sesión de exploración, se creará un nuevo token generado al azar para instalar las versiones actualizadas de Microsoft Edge.  El token no está asociado a ninguna información personal y solo se usa para el proceso de instalación y actualización, y para mejorar el servicio del actualizador.  

Microsoft Edge hace ping al servicio del actualizador de Microsoft Edge sobre el progreso de la instalación y la actualización.  Si se produce un error de instalación o actualización y está activado el informe de bloqueo, se creará un registro y se enviará a Microsoft.  Para más información sobre el envío de informes de bloqueos a Microsoft, vea la sección [Bloqueos](#crashes).  Microsoft recopila información acerca de cómo ha descargado Microsoft Edge, el éxito de la instalación y cualquier desinstalación para comprender mejor el éxito de las descargas de Microsoft Edge.  

Las actualizaciones automáticas están activadas de forma predeterminada para todos los usuarios de Microsoft Edge.  En todas las plataformas, Microsoft Edge comprueba si hay actualizaciones al inicio y periódicamente cuando se ejecuta.  En dispositivos Mac OS, Microsoft AutoUpdate comprueba también si hay actualizaciones de productos de Microsoft de forma periódica.  Hay disponibles controles y configuraciones adicionales para organizaciones.  Para más información sobre controles y configuraciones adicionales, consulte [Actualizar][DeployedgeUpdatePoliciesUpdate].  

## Modo Internet Explorer  

Microsoft Edge ofrece una experiencia simplificada con la integración de Internet Explorer \(IE\).  Microsoft Edge solo admite Internet Explorer 11, y el modo IE solo está disponible en Windows.  La característica del modo IE está disponible para las organizaciones mediante directivas de grupo.  El administrador opta por abrir un determinado sitio en modo IE en Microsoft Edge.  

:::image type="complex" source="./media/whitepaper-media/ie-mode.png" alt-text="Modo IE" lightbox="./media/whitepaper-media/ie-mode.png":::
   Modo IE  
:::image-end:::  

Microsoft Edge descarga la lista de sitios de una ubicación definida por el administrador mediante una directiva y almacena en caché el archivo que determina qué sitios deben abrirse en el modo IE.  Dependiendo de la configuración de Windows o de Internet Explorer 11, Microsoft Edge recopila datos de diagnóstico sobre el uso del modo IE (como los sitios que los usuarios visitan, los datos de rendimiento y confiabilidad, y los datos de uso de características).  En Windows 10, los datos de diagnóstico se recopilan según la configuración de datos de diagnóstico de Windows.  En Windows 8.1, se recopila información del sitio web si el usuario se ha suscrito a la característica Pasar de página y Sitios sugeridos en IE.  Es posible que la configuración de la recolección de datos en el modo IE no coincida con la configuración de privacidad de Microsoft Edge.  

Si el administrador activó la Detección de sitio empresarial, los datos del historial de exploración se recopilan periódicamente para ayudar a los administradores a revisar los sitios que los usuarios visitan y asegurar que todas las actualizaciones del sistema siguen admitiendo esos sitios.  Para más información sobre la Detección de sitio empresarial en IE11, vea [Recopilar datos mediante la Detección de sitio empresarial][InternetExplorer11DeployGuideCollectDataEnterpriseSiteDiscovery].  

Los usuarios que no sean de empresa en dispositivos Windows también pueden tener acceso al modo IE.  Para activar el modo IE, vaya a `edge://settings/defaultBrowser` y seleccione la opción **Permitir que los sitios se vuelvan a cargar en el modo de Internet Explorer**.  Para abrir las pestañas en modo IE, abra **Configuración y más (...)** > **Más herramientas** y seleccione **Volver a cargar en el modo de Internet Explorer**.  Después de activar el modo IE, Microsoft Edge pide periódicamente una lista de sitios no compatibles de un servicio de Microsoft.  La solicitud se enviará por HTTPS y no contendrá ningún identificador.  

Los datos de exploración de Internet Explorer se almacenan de forma local en Microsoft Edge y en Internet Explorer.  Para eliminar los datos de exploración mientras navega en el modo IE, vaya a `edge://settings/privacy` y borre los datos de **desactive la búsqueda de datos** y **quitar datos de búsqueda para Internet Explorer**.  


## Anuncios intrusivos  

Para mejorar la experiencia de navegación, Microsoft Edge ofrece la posibilidad de bloquear la carga de anuncios en sitios que muestran anuncios intrusivos o engañosos.  Cuando el bloqueo de anuncios está activado, Microsoft Edge descarga periódicamente de los servidores de Microsoft la lista de sitios más recientes donde se muestran anuncios intrusivos o engañosos y los almacena de forma local en el dispositivo.  No se incluyen identificadores de usuario en la solicitud de descarga.  Si visita un sitio de la lista, Microsoft Edge bloquea todos los anuncios del sitio y debería ver el mensaje `Ads blocked`.  Para permitir anuncios para el sitio, vaya a `edge://settings/content/ads` y cambie la configuración.  Excepto por la descarga de la lista de sitios con anuncios intrusivos, la característica de bloqueo de anuncios no envía información adicional a Microsoft ni solicita información adicional de Microsoft mientras explora la Web.  

## Listas de accesos directos  

La Lista de accesos directos en Microsoft Edge le permite encontrar fácilmente los sitios cerrados recientemente al pasar el icono de Microsoft Edge en la barra de tareas y abrir un menú contextual \(haga clic con el botón derecho en\).  Las tres últimas pestañas cerradas se almacenan de forma local para cada perfil.  Para eliminar sitios de la Lista de accesos directos de Windows 10, desplace el puntero en cada sitio y abra el menú contextual \(haga clic con el botón derecho\).  Para activar o desactivar la visualización de las pestañas de la lista de accesos directos recientemente, vaya a `edge://settings/privacy` y seleccione el **Elegir lo que quiere quitar cada vez que cierre el explorador** Configuración.  Al usar una ventana de InPrivate, Microsoft Edge no agrega información de la pestaña cerrada a la lista de accesos directos.  Cuando se usa el modo de invitado, la lista de accesos directos no está disponible.  Para más información sobre cómo eliminar datos de exploración, vea [Ver y eliminar el historial de exploración en Microsoft Edge][MicrosoftSupport10607].  

## Hora de red  

Microsoft Edge usa un servicio de hora de la red de Microsoft para realizar un seguimiento de la hora desde un origen externo, como un servidor de hora.  A intervalos aleatorios o cuando Microsoft Edge encuentra un certificado SSL expirado, Microsoft Edge puede enviar solicitudes a Microsoft para obtener la hora de un origen de confianza.  Las solicitudes se ejecutan con más frecuencia si Microsoft Edge detecta que el reloj del sistema no es preciso.  La inprecisión del reloj del sistema sucede si el usuario cambia la hora en el sistema operativo y entra en conflicto con la zona horaria correcta.  El servicio de hora de la red de Microsoft se usa para obtener la hora universal coordinada \(UTC\).  Las solicitudes no contienen cookies ni identificadores de usuario, y no se registran datos.

Personalice la apariencia de la página de pestaña nueva seleccionando el botón Personalizar.  Las preferencias de la página de nueva pestaña se establecen para cada perfil y se almacenan de forma local en el dispositivo. Las preferencias no se sincronizan entre dispositivos.

## Al iniciarse  

Microsoft Edge le permite desplazarse por donde lo dejó abriendo las últimas pestañas abiertas de la sesión de exploración anterior, incluidas las cookies de sesión, permanece disponibles en el inicio para restaurar las pestañas de la sesión anterior y mantenerse al día con los sitios que visitó.  Para habilitar la apertura de las pestañas de la última vez que se abrió en la sesión de exploración anterior, vaya a `edge://settings/onStartup` y active **Continuar, donde lo dejó**.  Si borra los datos de exploración cada vez que cierra el explorador, pero seleccionó la opción **Continuar donde lo dejó**, se eliminan los datos que especificó, pero la dirección URL continúa para la siguiente sesión.  

También puede configurar Microsoft Edge para abrir páginas específicas al inicio.  Las páginas que especifica se almacenan de forma local en el dispositivo y son específicas para cada perfil.  Si ha activado la sincronización de la Configuración, las páginas especificadas se sincronizan en todas las versiones de Microsoft Edge donde haya iniciado sesión.  Para habilitar la sincronización de la configuración, vaya a `edge://settings/profiles/sync` **Configuración** para activarla.  

Las pestañas del modo InPrivate y de invitado no se restauran al inicio.  

## Supervisión de contraseña
Microsoft Edge está comprometido a mantener su seguridad en la Web.  Para ayudarle a mantener la privacidad y seguridad de su información personal, si inicia sesión en Microsoft Edge, el Monitor de contraseñas le avisará si las credenciales se han expuesto en alguna infracción de datos de terceros.  Si el Monitor de contraseñas está activado, se aplican un algoritmo hash a las credenciales guardadas y se cifran de forma local en el dispositivo, se envían a servidores de Microsoft a través de HTTPS y se comparan con una lista de credenciales de en infracciones de datos conocidas.  El identificador de la cuenta con la sesión iniciada se envía de forma segura junto con las credenciales cifradas y en hash al servicio de Monitor de contraseñas.  Si se encuentra una credencial en la lista de credenciales en infracciones de datos conocidas, Microsoft envía una respuesta cifrada a su versión de Microsoft Edge para advertirle de que sus credenciales se han detectado como parte de un ataque de hackers o de una infracción de seguridad.  No se almacenan datos en servidores de Microsoft una vez completada la comprobación.   

La característica solo está disponible para los usuarios que hayan iniciado sesión en Microsoft Edge.  Microsoft Edge pide permiso para activar el Monitor de contraseñas.  Para activar o desactivar el Monitor de contraseñas, vaya a `edge://passwords`.

## Pagos  

Microsoft Edge le ayuda a ser más productivo ya que le permite guardar la información de pago en su perfil del explorador y le ofrece la opción de rellenar automáticamente formularios de pago con la información cuando lo necesite mientras navega.  Cuando se encuentra en un formulario de pago similar, Microsoft Edge ofrece rellenar el formulario con la información guardada.  La información de tarjeta de crédito y otra información de pago solo se guarda con su permiso explícito.  

Microsoft Edge le pregunta si quiere almacenar la información de pago si está activada la función Autorrellenar de pago.  La información se cifra de forma local en el dispositivo.  Para eliminar la información de pago guardada, vaya a `edge://settings/payments`.  Al eliminar la información de pago guardada, la información ya no aparecerá como una sugerencia de Autorrellenar.  Para no guardar ninguna información de pago, vaya a `edge://settings/payments` y desactive la característica.  

Microsoft Edge admite la API PaymentRequest al permitirle pagar las compras con la información de pago que haya guardado previamente con Autorrellenar.  La API de PaymentRequest permite que el comerciante solicite la siguiente información: número de tarjeta de crédito, caducidad de la tarjeta de crédito, nombre completo, dirección de facturación, dirección de correo electrónico, número de teléfono y dirección de envío.  La API indica al comerciante que el usuario tiene información de tarjeta de crédito guardada, pero no comparte información con el comerciante hasta que usted lo acepte.  Para desactivar la característica de pagos, vaya a `edge://settings/privacy`.  

Si ha guardado previamente información de pago en su cuenta de Microsoft, también está disponible para Autorrellenar en el explorador.  La información de pago almacenada en su cuenta de Microsoft se sincroniza entre dispositivos.  Si ha realizado anteriormente alguna compra de Xbox o Microsoft Store, es posible que ya tenga información de pago guardada en su cuenta de Microsoft.  Durante el relleno automático del pago, una tarjeta de la cuenta de Microsoft se enmascara y solo se muestra por completo después de la autenticación de dos factores.  El enmascaramiento ofrece seguridad adicional mientras se recupera la información de pago.  

## Personalizar publicidad, búsqueda, noticias y otros servicios Microsoft  

Si permite la personalización, el equipo de Microsoft Edge recopila y usa el historial de navegación de Microsoft Edge para personalizar las experiencias y la publicidad en [Bing][|::ref3::|Main], Microsoft News y otros servicios de Microsoft.  La personalización ofrece resultados de búsqueda, publicidad y noticias más relevantes y útiles.  Por ejemplo, si el equipo de Microsoft Edge deduce basándose en su navegación que prefiere comprar en una determinada tienda, puede que los anuncios que vea estén adaptados a los productos de la tienda específica.  De forma similar, si visita a menudo blogs de viajes y lee artículos de viajes, el suministro de noticias puede incluir más contenido de noticias relevantes sobre viajes.  
 
La característica solo está disponible para los usuarios que tengan una cuenta de Microsoft que no sea infantil.  La característica no está disponible para los usuarios que hayan iniciado sesión en Microsoft Edge con una cuenta profesional o educativa.  

El historial de exploración se recopila y usa únicamente para la personalización si se cumplen las cuatro condiciones siguientes:  

*   El usuario inicia sesión en una cuenta de Microsoft que no es infantil. 
*   El usuario ha concedido permiso para la recopilación y el uso de los datos para la personalización.  
*   Las directivas de grupo de usuario administradas por la organización \(empresa, escuela, etc.) permiten la personalización.  
*   El usuario no usa el explorador en el modo de invitado o en InPrivate.  

El historial de exploración y otros datos relevantes se transfieren a través de HTTPS y se adjuntan a la información de su cuenta de Microsoft.  El historial de exploración se almacena en servidores seguros de Microsoft.  Puede ver y eliminar el historial de exploración compartido anteriormente yendo al [panel de privacidad de Microsoft][MicrosoftAccountPrivacy].  El historial de exploración se almacena en servidores de Microsoft seguros hasta 45 días.  Después de 45 días los datos se eliminan y no se usan para la personalización.  

Puede modificar sus intereses o dejar de participar en anuncios personalizados desde la [configuración de publicidad][MicrosoftAccountPrivacyAdSettings] en el [panel de privacidad de Microsoft][MicrosoftAccountPrivacy].  

> [!NOTE]
> Optar por no tener anuncios personalizados en el [panel de privacidad de Microsoft][MicrosoftAccountPrivacy] no desactiva la recopilación y el uso del historial de exploración para personalizar los resultados de búsqueda y el contenido de la fuente de noticias.  Para deshabilitar la recopilación y el uso del historial de navegación de Microsoft Edge para obtener resultados de búsqueda personalizados y noticias, vaya a `edge://settings/privacy` y en **Personalice su experiencia web**, desactive **Mejore su experiencia web al permitir que Microsoft use el historial de exploración de esta cuenta para personalizar la publicidad, la búsqueda, las noticias y otros servicios Microsoft**.  Si detiene el uso compartido de los datos, Microsoft ya no recopilará ni usará el historial de exploración para personalizar anuncios, resultados de búsqueda y noticias. Para más información sobre la personalización en Microsoft Edge, vea [Historial de exploración de Microsoft Edge para publicidad y experiencias personalizadas][MicrosoftSupport4532583].  

## Imprimir  

Microsoft Edge le permite imprimir páginas web, archivos PDF u otros contenidos con una variedad de dispositivos y aplicaciones.  Al imprimir en una impresora, aplicación o PDF, Microsoft Edge envía los comandos y la información de archivo al sistema operativo de su dispositivo.  La información no se enviará a Microsoft, y todos los datos que se envíen al sistema operativo de su dispositivo para imprimir se eliminarán inmediatamente después de que se complete o cancele la impresión.  Para cambiar el destino de impresión, vaya a `edge://settings/printing`.  

También puede imprimir páginas web y archivos en un archivo PDF con la función Imprimir en PDF de Microsoft, que no envía ningún dato sobre el archivo a Microsoft.  Cualquier anotación que se realice en el archivo PDF se guarda de forma local en el archivo.  

## Perfiles  

Los perfiles de Microsoft Edge permiten separar los datos de navegación en perfiles independientes.  Los datos asociados a un perfil son independientes de los datos asociados a otros perfiles.  Los favoritos personales y el historial, por ejemplo, no se sincronizan con su cuenta profesional si configura cada uno en perfiles distintos.  

Sin embargo, los usuarios pueden cambiar fácilmente entre perfiles existentes en Microsoft Edge sin necesidad de contraseñas.  Si los usuarios tienen acceso al mismo dispositivo, los usuarios pueden crear otro perfil en la misma versión de Microsoft Edge sin el permiso del propietario del perfil actual.  Al quitar el perfil de la configuración de Microsoft Edge, se eliminan de forma permanente los datos de exploración del perfil específico almacenados en el dispositivo, como el historial de exploración, favoritos, datos de relleno de formularios y contraseñas.  Los datos sincronizados con su cuenta aún pueden almacenarse en la nube de Microsoft y es posible eliminarlos desde el [panel de privacidad de Microsoft][MicrosoftAccountPrivacy].  

El modo de invitado es una instancia temporal de un perfil nuevo.  Permite buscar en el dispositivo de otro usuario sin modificar el perfil que ha iniciado sesión.  Los datos de exploración del modo de invitado, como favoritos, el historial de exploración, las contraseñas y los datos de relleno de formularios, no se conservan después de cerrar todas las ventanas del modo de invitado.  Los archivos descargados se almacenan en el dispositivo, pero se elimina el historial de las descargas.  El modo de invitado le permite explorar la Web sin tener que iniciar sesión en otros sitios automáticamente.  Microsoft Edge no envía a los sitios web ninguna información para indicar que el usuario está navegando en modo de invitado.  Cuando usa el modo de invitado, el permiso para recopilar datos de diagnóstico sobre cómo usa el explorador y los sitios web que visite se tomará del perfil de Microsoft Edge desde el que se inició la sesión en modo de invitado.  Todos los datos de exploración de la sesión de modo invitado específica se eliminan después de cerrar todas las ventanas de invitado.  

La exploración de InPrivate es el modo de exploración privada en el que no se recuerdan el historial de exploración, el historial de descargas, las cookies y datos del sitio, y los datos de relleno de formularios.  Microsoft Edge guarda los archivos descargados, así como cualquier otro favorito creado cuando se explora en InPrivate.  De forma predeterminada, al navegar en InPrivate, Microsoft no recopila información acerca de los sitios web que visite para propósitos de mejora del producto.  Es posible que su escuela, lugar de trabajo o su proveedor de servicios de Internet aún puedan ver su actividad de navegación.  Los datos de exploración para la sesión de InPrivate específica se eliminan cuando se cierran todas las ventanas de InPrivate.  Cuando se usa el teclado del Editor de métodos de entrada \(IME\) de Windows para escribir y usar la entrada de lápiz, se pueden recopilar datos para mejorar el reconocimiento del idioma y la funcionalidad de las sugerencias.  Para que Microsoft deje de recopilar los datos escritos en el teclado o introducidos con la entrada de lápiz al usar el teclado del IME de Windows en ventanas de búsqueda normales y en InPrivate, vaya a **Inicio** > **Configuración** > **Privacidad**y seleccione **Personalización de entrada manuscrita y escritura**.  Para más información sobre la exploración de InPrivate, vea [Navegar en InPrivate en Microsoft Edge][MicrosoftSupport4533513].  
  

## Leer en voz alta  

Microsoft Edge ofrece Leer en voz alta, que lee el contenido de una página web para el usuario.  Para empezar a leer en voz alta, mueva el puntero en cualquier lugar de la página y abra el menú contextual \ (haga clic con el botón derecho \) o abra **Configuración y más (...)** y seleccione **Leer en voz alta**.  Leer en voz alta ofrece varias voces que se pueden usar para leer el contenido de la página web.  Si usa voces que están [instaladas en Windows 10][OfficeSupport4c83a8d8748642f78e462b0fdf753130] en la sección **Hora e idioma** de la configuración de Windows 10, todas las voces que usó previamente se almacenan en la memoria caché local y se pueden borrar en `edge://settings/clearBrowserData`.  

Cuando empieza a leer en voz alta, Microsoft Edge usa la [API de Web Speech][GithubW3cIncubatorCommunityGroupSpeechApi].  Según la voz que seleccione, el contenido de la página se convierte de texto a voz con una biblioteca del lado cliente proporcionada por la plataforma \(por ejemplo, una específica del sistema operativo\), o una biblioteca del servidor con tecnología de Microsoft Cognitive Services.  Si el contenido se convierte a voz con una biblioteca del lado cliente, no se envía información a los servidores de Microsoft.  Si el contenido se convierte a voz con Microsoft Cognitive Services \(indicado con la palabra "Online" en cualquiera de los nombres de voz\), el texto, junto con un token generado aleatoriamente, se enviará a Microsoft.  Una vez completada la conversión, el servicio devuelve el texto narrado en un archivo de audio al dispositivo.  Todos los datos se cifran a medida que se transfieren desde el dispositivo a Microsoft, y viceversa.  El texto que se envía a Microsoft y el archivo de audio generado se eliminan de forma inmediata cuando se produce la conversión. No se almacenan otros datos sobre el contenido web durante ningún período de tiempo.

Como parte del aprendizaje, algunos usuarios podrían experimentar nuevas funciones antes de que estén disponibles para todos.  Para habilitar la nueva funcionalidad para usuarios seleccionados al azar, Microsoft Edge envía de forma periódica información necesaria sobre su sistema operativo, canal, versión, país o región y otros datos de configuración del dispositivo al servicio de configuración de Microsoft Edge.  Los datos se envían con un identificador reconfigurable único para su explorador.  Los datos se transmiten al servicio a través de HTTPS.  Los datos se usan para recibir actualizaciones que permiten habilitar nuevas funciones, mantener Microsoft Edge actualizado y funcionando correctamente, y mejorar los productos y servicios Microsoft. Para más información sobre los controles y configuraciones adicionales para organizaciones, consulte [Configuraciones y experimentación de Microsoft Edge][DeployedgeConfigurationExperiments].

Para comprender cómo afecta la nueva funcionalidad a Microsoft Edge y a los servicios Microsoft, Microsoft Edge envía un identificador configurable único al explorador y una etiqueta de funcionalidad que codifica las nuevas funcionalidades que se han habilitado para Microsoft Edge y servicios Microsoft. ## Solucionar errores de navegación  

Si Microsoft Edge detecta tiempos de espera de conexión SSL, errores de certificado u otros problemas de red que pueden deberse a un portal cautivo \(por ejemplo, una red Wi-Fi en un hotel o aeropuerto\), Microsoft Edge envía una solicitud a `http://edge.microsoft.com/captiveportal/generate_204` y comprueba el código de respuesta.  Si la solicitud se redirige a otra URL, Microsoft Edge abre la dirección URL en una nueva pestaña, suponiendo que sea una página de inicio de sesión.  Las solicitudes a la página de detección del portal cautivo son un servicio sin estado, las solicitudes no se registran y las cookies no se envían ni se guardan.  En las plataformas Windows, Microsoft Edge usa un servicio de portal cautivo de Windows.  En caso contrario, se usa el servicio de portal cautivo de Microsoft Edge.  Para desactivar el servicio, vaya a `edge://settings/privacy` y desactive la opción **Usar un servicio web para ayudar a resolver los errores de navegación**.  

## Inicio de sesión e identidad  

Iniciar sesión en Microsoft Edge ofrece funciones adicionales para que el explorador sea más productivo para usted.  Para iniciar sesión de forma sencilla, al iniciar Microsoft Edge por primera vez, se intenta detectar la identidad del sistema operativo.  Si Microsoft Edge detecta su identidad desde el sistema operativo pero no desea que permanezca iniciada su sesión en Microsoft Edge, vaya a `edge://settings/profiles` y cierre la sesión o quítela.  Para iniciar sesión en Microsoft Edge, si la identidad no se detecta desde el sistema operativo, vaya a `edge://settings/profiles`.  

Si se agrega una identidad nueva al sistema operativo y su perfil de Microsoft Edge no tiene actualmente una identidad, Microsoft Edge agrega la identidad específica a su perfil.  Si inicia sesión en Microsoft Edge con una cuenta de Microsoft o una cuenta profesional o educativa y no tiene una identidad en su perfil de Windows 10, la cuenta específica se agregará a su perfil de Windows 10, a menos que elija específicamente no agregarla a Windows 10 mientras inicia sesión.  

El perfil que ha iniciado sesión no empieza a sincronizar los datos sin su permiso explícito mientras inicia Microsoft Edge por primera vez o al iniciar sesión en el explorador.  

El inicio de sesión en Microsoft Edge permite el inicio de sesión único \(inicia sesión automáticamente en ciertos sitios web como Bing\) y otras experiencias basadas en la identidad, como la sincronización. Si solo quiere iniciar sesión en Microsoft Edge y no en otros sitios de Microsoft (como [Bing][|::ref4::|Main]), puede cerrar la sesión del sitio específico.  Microsoft Edge crea una cookie de cierre de sesión que indica a Microsoft Edge que no inicie sesión en el sitio específico para futuras visitas.  Para volver a iniciar sesión en sitios específicos con su nombre de usuario y contraseña, o bien puede eliminar las cookies, vaya a `edge://settings/privacy`.  Para más información sobre cómo eliminar datos de navegación, vea [Ver y eliminar el historial del explorador en Microsoft Edge][MicrosoftSupport10607].  

Para impedir que se asocie cualquier identidad con Microsoft Edge, quite el perfil de Microsoft Edge o cierre sesión en Microsoft Edge.  Para eliminar todos los datos asociados con su perfil de Microsoft Edge del dispositivo, debe quitar su perfil de Microsoft Edge.  Al eliminar todos los datos, no se eliminan los datos sincronizados previamente asociados a la identidad.  

Su identidad en Microsoft Edge en macOS se comparte entre aplicaciones de Microsoft.  Una identidad compartida le permite iniciar sesión en una aplicación de Microsoft sin tener que escribir las credenciales por separado si ha iniciado sesión en otra aplicación de Microsoft en el dispositivo.  En macOS, no se inicia sesión automáticamente en Microsoft Edge en función del estado de autenticación en otra aplicación de Microsoft.  Al intentar iniciar sesión en Microsoft Edge, ofrece usar las credenciales de otra aplicación de Microsoft en el dispositivo para iniciar sesión en Microsoft Edge de forma sencilla.  De forma similar, cuando haya iniciado sesión en una cuenta de Microsoft Edge, si intenta iniciar sesión en otras aplicaciones de Microsoft, puede usar las credenciales de Microsoft Edge para ayudarle a iniciar sesión en el resto de las aplicaciones de Microsoft en el dispositivo sin tener que volver a escribir sus credenciales.  

No se puede iniciar sesión en Microsoft Edge cuando se usa el modo de invitado o InPrivate.  

## SmartScreen  

SmartScreen está diseñado para ayudarle a navegar por la Web de forma segura.  Cuando va a sitios web o descarga archivos, SmartScreen comprueba la reputación de la dirección URL o el archivo.  Si SmartScreen determina que el sitio o el archivo son malintencionados, le impide ir al sitio o descargar el archivo.  

:::image type="complex" source="./media/whitepaper-media/smart-screen.png" alt-text="SmartScreen" lightbox="./media/whitepaper-media/smart-screen.png":::
   SmartScreen  
:::image-end:::  

Mientras navega por la web, SmartScreen categoriza los sitios web y las descargas como de mayor tráfico, peligrosos o desconocidos.  El tráfico principal son los sitios populares que SmartScreen ha determinado que son dignos de confianza.  Si visita un sitio marcado como peligroso, SmartScreen bloquea inmediatamente el acceso al sitio.  Cuando se dirija a un sitio desconocido, SmartScreen comprueba su reputación para determinar si debe acceder al sitio.  

SmartScreen usa tres tipos de comprobaciones de reputación.  En primer lugar, SmartScreen comprueba la URL de los sitios que visite en una lista local para determinar si el sitio forma parte del tráfico principal o si es un sitio peligroso conocido.  Cuando visita un sitio de tráfico principal, SmartScreen no envía la URL al servicio SmartScreen.  Si la dirección URL se encuentra en la lista local de sitios peligrosos, SmartScreen lo bloquea, lo que evita la carga de cualquier parte del contenido web malintencionado.  Microsoft Edge descarga periódicamente una lista actualizada de el tráfico principal y los sitios peligrosos en el dispositivo.  El segundo tipo de comprobación de URL es una comprobación de reputación asincrónica en el servicio SmartScreen.  SmartScreen realiza la comprobación de todas las direcciones URL que no están categorizadas como tráfico principal.  Microsoft Edge pasa la URL, información relevante sobre el sitio, un identificador único del dispositivo y la información de la ubicación general al servicio SmartScreen para determinar la seguridad del sitio.  La información proporcionada por Microsoft Edge permite al servicio identificar nuevos sitios peligrosos y mantenerse al día con las últimas amenazas de seguridad.  Los resultados de las comprobaciones de URL se almacenan de forma local en el dispositivo y se eliminan automáticamente al final de la sesión del explorador.  Todas las solicitudes al servicio SmartScreen se llevan a cabo con cifrado HTTPS.  

En tercer lugar, SmartScreen comprueba los archivos descargados para ayudar a evitar daños en el dispositivo.  SmartScreen realiza de forma síncrona una comprobación de la reputación de archivo binario cuando finaliza la descarga.  Microsoft Edge envía información sobre el archivo, como el hash del archivo, nombre de archivo, URI de descarga y un identificador único para el dispositivo a SmartScreen para realizar la comprobación de la reputación.  Todas las solicitudes de SmartScreen se llevan a cabo con cifrado HTTPS.  El servicio SmartScreen devuelve el resultado de la comprobación, lo que permite o no descargar completamente el archivo.  Los resultados se almacenan de forma local en el dispositivo.  

El servicio SmartScreen almacena datos sobre las comprobaciones de reputación y crea una base de datos de URL y archivos maliciosos conocidos.  Estos datos se almacenan en servidores seguros de Microsoft y sólo se utilizan para los servicios de seguridad de Microsoft.  Los datos no se usan nunca para identificarle ni marcarle de ninguna manera.  Borrar la memoria caché de exploración borra todos los datos de la URL de SmartScreen almacenada de forma local.  Borrar el historial de descargas eliminará cualquier dato almacenado de forma local de SmartScreen sobre las descargas de archivos.  

SmartScreen está activado de forma predeterminada para Microsoft Edge.  Para deshabilitar SmartScreen, vaya a `edge://settings/privacy` y, en **Servicios** desactive la opción **SmartScreen de Microsoft Defender**.  Esta configuración es la misma para todos los perfiles asociados a la instalación de Microsoft Edge en el dispositivo.  La configuración no está sincronizada entre dispositivos.  La configuración se aplica a la navegación InPrivate y al modo de invitado.  Si el dispositivo se administra con directivas de grupo establecidas por su organización, la configuración se refleja en Microsoft Edge.  Para ver la configuración, vaya a `edge://settings/privacy`.  Para más información sobre SmartScreen, consulte [SmartScreen: Preguntas más frecuentes][MicrosoftSupport17443].  

De forma opcional, SmartScreen comprueba las direcciones URL de los archivos que descarga para ver si alguno se ha clasificado como clasificada como aplicación potencialmente no deseada.  Bloquear las aplicaciones potencialmente no deseadas le ayudará a proporcionar experiencias de Windows más productivas, con mejor rendimiento y agradables.  La configuración está desactivada de forma predeterminada y solo está disponible en dispositivos con Windows 10.  Para habilitar la característica, vaya a `edge://settings/privacy` y active la opción **Bloquear aplicaciones potencialmente no deseadas**.  Para más información sobre cómo se categorizan las aplicaciones potencialmente no deseadas, consulte [Aplicación potencialmente no deseada (PUA)][WindowsSecurityThreatProtectionIntelligenceCriteriaPotentiallyUnwanted].  Para más información sobre cómo configurar la opción de configuración, vea [Detectar y bloquear aplicaciones potencialmente no deseadas][WindowsSecurityThreatProtectionWindowsDefender].  

## Reconocimiento de voz

Para convertir el discurso en texto, Microsoft Edge es compatible con la [API de Web Speech][GithubW3cIncubatorCommunityGroupSpeechApi].  Si un sitio web incluye una característica web que requiere capturar y traducir su voz a texto y solicita acceso al micrófono, Microsoft Edge envía el audio capturado a un servicio de Microsoft en el que se traduce en texto.  El audio grabado se envía con un token generado aleatoriamente a través de una conexión HTTPS segura a los servicios cognitivos de Microsoft Azure.  El contenido de audio grabado no se almacena para ningún fin.  El texto se vuelve a enviar al dispositivo y, a continuación, se envía al sitio Web.  

Para desactivar la voz traducida al texto, puede denegar el acceso del micrófono de cualquier sitio en el que se pida permiso.  Para desactivar el permiso del micrófono para todos los sitios, vaya a `edge://settings/content/microphone`.  

## Revisión ortográfica  

Microsoft Edge revisa la ortografía mientras escribe en el explorador.  El servicio de corrección ortográfica se completa de forma local en el dispositivo. Microsoft Edge no envía información sobre el texto escrito a Microsoft para la revisión ortográfica.  Para desactivar la característica, vaya a `edge://settings/languages` y en **Revisar ortografía** desactive la configuración para cada idioma.  

Cuando agrega un nuevo idioma a Microsoft Edge, el navegador descarga el diccionario para el nuevo idioma en el dispositivo mediante HTTPS.  El diccionario se usa para el servicio de revisión de ortografía integrado.  Al eliminar el idioma de la configuración de Microsoft Edge, se elimina el diccionario del dispositivo.  El modo de invitado no usa el diccionario personalizado del perfil o cualquier idioma agregado.  Para agregar o quitar palabras en el diccionario local, vaya a `edge://settings/languages` y, en **Revisar ortografía** seleccione **Agregar o eliminar palabras**.  

## Sugerir sitios similares  

Para ayudar a resolver los errores de escritura de URL en la barra de direcciones que provocan un error en el sitio web, es posible que Microsoft Edge recomiende una URL corregida.  Cuando se produce un error de navegación de un sitio web, Microsoft Edge envía el dominio de la dirección web al servicio de Microsoft para sugerir una URL corregida.  Microsoft Edge no incluye identificadores o tokens en el dominio.  Si el servicio encuentra una sugerencia, devuelve la URL sugerida.  Microsoft almacena el dominio incorrecto y el dominio sugerido para ayudar a mejorar el servicio.  Para ayudarle a desplazarse a los sitios correctos, la característica está activada de forma predeterminada.  Para desactivar la característica, vaya a `edge://settings/privacy` y, en **Servicios**, deshabilite la opción **Sugerir sitios similares cuando no se puede encontrar un sitio web**.  

## Sincronización  

Iniciar sesión en Microsoft Edge con una cuenta de Microsoft le permite sincronizar los datos de exploración en todas las versiones con sesión iniciada de Microsoft Edge.  Puede sincronizar los favoritos, la configuración, los datos de relleno de formularios \(direcciones y más\), contraseñas, extensiones y colecciones.  Debe permitir activar la sincronización en Microsoft Edge y todos los tipos de datos sincronizados se pueden activar o desactivar por separado.  Favoritos contiene todas las pestañas que ha reservado anteriormente en la versión heredada de Microsoft Edge, que se sincronizan con el resto de los favoritos.  Los favoritos u otros datos eliminados o modificados de una versión con sesión iniciada de Microsoft Edge se sincronizan con todas las versiones con sesión iniciada de Microsoft Edge en las que la sincronización está activada.  Para administrar las configuraciones de sincronización, vaya a `edge://settings/profiles/sync`.  Su organización puede administrar la configuración de sincronización.

:::image type="complex" source="./media/whitepaper-media/sync.png" alt-text="Sincronización" lightbox="./media/whitepaper-media/sync.png":::
   Sincronización  
:::image-end:::  

Para que la sincronización funcione, se envían los datos de conectividad y configuración del dispositivo adicionales necesarios para ofrecer la experiencia de sincronización, como el nombre del dispositivo, la marca y el modelo.  Es posible que se eliminen los datos de del [panel de dispositivos de Microsoft][MicrosoftAccountDevices].  Para administrar sus favoritos sincronizados, vaya a `edge://favorites`.  Para administrar el resto de los tipos de datos, vaya a `edge://settings/profiles`.  

Todos los datos sincronizados se cifran en tránsito mediante HTTPS cuando se transfieren entre el explorador y los servidores de Microsoft.  Los datos sincronizados también se almacenan en un estado cifrado en servidores de Microsoft.  Los tipos de datos confidenciales, como las direcciones y las contraseñas, se cifran en el dispositivo antes de sincronizarse.  Si usa una cuenta profesional o educativa, se cifran aún más todos los tipos de datos antes de sincronizarlos con Microsoft Information Protection.  El resto de los tipos de datos sincronizados se almacenan hasta que elimine los datos, se elimine la cuenta o la cuenta se vuelva inactiva.  Un identificador de cuenta se adjunta a todos los datos sincronizados, ya que el identificador es necesario para realizar la sincronización entre varios dispositivos. 

Los datos de exploración del modo invitado y de InPrivate no se sincronizan con su cuenta de Microsoft.  Sin embargo, los favoritos creados durante sesiones de InPrivate se sincronizan en las versiones con sesión iniciada de Microsoft Edge en que la sincronización está activada.  

## Prevención de seguimiento  

Microsoft Edge está diseñado para detectar y bloquear rastreadores conocidos.  Los usuarios pueden elegir entre tres niveles de prevención de rastreo: Básica, Equilibrada y Estricta.  Para proteger la privacidad del usuario, Equilibrada está seleccionada de forma predeterminada.  Microsoft Edge detecta los rastreadores antes de cargarlos en la página con una lista de código abierto de rastreadores conocidos.  La lista se descarga al dispositivo periódicamente a medida que se actualiza la lista.  El número de rastreadores bloqueados y los nombres de dichos rastreadores se almacenan de forma local en el dispositivo para propósitos estadísticos.  Para borrar los datos, vaya a `edge://settings/privacy/blockedTrackers`.  La detección y el bloqueo de rastreadores se produce de forma local en el dispositivo.  Para deshabilitar la prevención de seguimiento, vaya a `edge://settings/privacy`.  Para más información sobre la Prevención de seguimiento, consulte [Obtener información sobre la prevención de seguimiento en Microsoft Edge][MicrosoftAccount4533959].  

Para desactivar las actualizaciones de listas, puede usar la directiva de grupo siguiente, [Habilitar las actualizaciones de componentes en Microsoft Edge][DeployedgePoliciesComponentupdatesenabled].

## Traducir  

En Microsoft Edge, puede explorar la web y traducir las páginas web en el idioma que prefiera.  La característica de traducción integrada usa un servicio en el dispositivo que muestrea partes aleatorias de una página web para detectar el idioma original.  Si el idioma detectado no es uno de los idiomas predeterminados, Microsoft Edge ofrece traducir la página web a su idioma de visualización u otro idioma que elija.  Microsoft Edge no traduce una página web sin su permiso.  Para traducir una página web, también puede hacer clic con el botón derecho en la página y seleccionar **Traducir**.  Para traducir la página web, Microsoft Edge envía el contenido de la página y un token generado al azar al servicio de traducción de Microsoft Azure a través de una conexión HTTPS segura.  El contenido de la página no se almacena para ningún fin.  Para que Microsoft Edge deje de ofrecer traducir las páginas web, vaya a `edge://settings/languages` y desactive la opción **Preguntar si quiero traducir páginas web que no están escritas en un idioma que entiendo**.  

## Aplicaciones web y sitios anclados  

Microsoft Edge permite instalar aplicaciones web creadas por desarrolladores de sitios web y anclar sus sitios favoritos.  

Cuando ancla un sitio, se agrega a la barra de tareas o al dock.  Los datos se almacenan de forma local en el dispositivo.  En el caso de algunos sitios, se comparte con el sitio información acerca de si el sitio se ha anclado, para que el sitio no solicite que se ancle.  Puede administrar los sitios anclados en la barra de tareas o en el dock.  Los sitios anclados se abren en ventanas de Microsoft Edge y usan los mismos permisos de sitio y configuración de datos de diagnóstico que la versión específica de Microsoft Edge.  

## WebView

Microsoft Edge WebViews permite que los desarrolladores de aplicaciones hospeden contenido web en aplicaciones nativas de Windows 10 y Windows 7.  Las aplicaciones que hospedan Microsoft Edge WebView pueden enviar a Microsoft datos de diagnóstico, como, por ejemplo, el uso de la plataforma web de Microsoft Edge y los sitios que visite en la aplicación.  Para habilitar la recopilación de datos de diagnóstico, vaya a `edge://settings/privacy` y en **Ayudar a mejorar Microsoft Edge**, active la opción.  Para desactivar la recopilación de datos de diagnóstico de Microsoft Edge en Windows 10, vaya a **Inicio** > **Configuración** > **Privacidad**y seleccione **Comentarios y diagnósticos**.  Para desactivar la recopilación de datos de diagnóstico para todas las demás plataformas, vaya a `edge://settings/privacy` en una sesión de exploración normal y desactive las opciones **Mejorar los productos de Microsoft enviando informes de bloqueo y datos sobre cómo usa el explorador** y **Mejore las búsquedas y los productos de Microsoft enviando información acerca de los sitios web que visita en Microsoft Edge**.  Las aplicaciones que hospedan Microsoft Edge WebView pueden recopilar otros datos controlados por la administración de recopilación de datos del desarrollador y las directivas de privacidad relevantes.  

## Sitios web que visite para mejorar el producto y el servicio  

Si eligió mejorar las búsquedas y otros productos y servicios de Microsoft, Microsoft Edge envía información acerca de los sitios web que visita a Microsoft.  Los datos de diagnóstico se usan en conjunto para mejorar los productos y servicios Microsoft para todos los usuarios y no se recopilan ni almacenan con su cuenta de Microsoft.  El envío de información acerca de los sitios que visite en Microsoft Edge ayuda a Microsoft a entender la rapidez con que los sitios se cargan y aumenta la relevancia de los resultados de búsqueda.  

Entre los datos de diagnóstico se incluyen la dirección URL de la página que visite, métricas del sitio web, el título de la página, cómo accedió a la página, información sobre el contenido de la página y otra información relevante sobre la navegación de la página.  Algunos datos de diagnóstico se adjuntan con un identificador único para el dispositivo.  En caso contrario, los datos de diagnóstico se asocian a un identificador reconfigurable único para el explorador.  Los identificadores no contienen información personal.  El equipo de Microsoft Edge respeta la confidencialidad de los datos de diagnóstico al restringir el acceso a los datos y/o depurar la información de identificación personal.  Para restablecer el identificador único para el explorador en dispositivos con Windows 10, vaya a **Inicio** > **Configuración** > **Privacidad** > **Comentarios y diagnósticos** y, en > **Datos de diagnóstico**, cambie de **Completo** a **Básico**.  En otras plataformas, para generar un nuevo identificador reconfigurable para el explorador, vaya a `edge://settings/privacy` y cambie la opción **Mejore las búsquedas y los productos de Microsoft enviando información acerca de los sitios web que visita en Microsoft Edge** de **activado** a **desactivado**.  La funcionalidad de restablecimiento es diferente en los dispositivos administrados con directivas de grupo establecidas por la organización.  

En el caso de Microsoft Edge para Windows 10, la configuración está determinada por la configuración de datos de diagnóstico de Windows.  Para ver la configuración reflejada en Microsoft Edge, vaya a `edge://settings/privacy` en **Privacidad** y **Servicios**.  Para cambiar la configuración de Windows, abra **Inicio** > **Configuración** > **Privacidad** y seleccione **Comentarios y diagnósticos**.  En todas las demás plataformas que controlan la recopilación de datos de diagnóstico, vaya a `edge://settings/privacy` y active o desactive la opción **Mejore las búsquedas y los productos de Microsoft enviando información acerca de los sitios web que visita en Microsoft Edge**.  Esta configuración es la misma para todos los perfiles asociados a la instalación de Microsoft Edge en el dispositivo.  La configuración no está sincronizada entre dispositivos.  Si el dispositivo se administra con directivas de grupo establecidas por su organización, se refleja en `edge://settings/privacy`.  

Los datos de diagnóstico se cifran en tránsito y se almacenan en servidores de Microsoft seguros en un formato sin procesar durante hasta 30 días.  Después de desidentificar los datos de diagnóstico y agregarlos, se eliminarán los datos de diagnóstico sin procesar.  Los datos de diagnóstico desidentificados y agregados se almacenan durante un máximo de 2 años para seguir mejorando los productos y servicios Microsoft.  Como los datos de diagnóstico no se recopilan o almacenan con su cuenta de Microsoft, no es posible ver o eliminar los datos de diagnóstico desde el [panel de privacidad de Microsoft][MicrosoftAccountPrivacy].  En dispositivos con Windows 10, para solicitar la eliminación de los datos de diagnóstico asociados al dispositivo, vaya a **Inicio** > **Configuración** > **Privacidad** > **Comentarios y diagnósticos** y, en **Eliminar datos de diagnóstico**, seleccione **Eliminar**.  En otras plataformas, los datos de diagnóstico se eliminan o se agregan por completo y anonimizan en un plazo de 30 días.  

Para ver la información de diagnóstico recopilada en Windows 10, vaya a **Inicio** > **Configuración** > **Privacidad** y seleccione **Comentarios y diagnósticos** y seleccione **Ver datos de diagnóstico**.  En otras plataformas, para ver los datos, vaya a `edge://data-viewer`.  En otras plataformas, para habilitar la característica Visor de datos de diagnóstico, vaya a `edge://settings/privacy` y active la opción **Mejore las búsquedas y los productos de Microsoft enviando información acerca de los sitios web que visita en Microsoft Edge**.  

Los datos de diagnóstico nunca se envían mientras se navega en InPrivate o en modo de invitado.  

## Protección de aplicaciones de Windows Defender  

Protección de aplicaciones de Windows Defender \(WDAG\) es una característica disponible para las organizaciones.  Cuando la Protección de aplicaciones de Windows Defender está activada, Microsoft Edge abre los sitios que no sean de confianza en un contenedor aislado para proteger los recursos de la organización frente a los sitios malintencionados o los ataques de suplantación de identidad.  La característica solo está activada con directivas de grupo que administra la organización y solo está disponible en versiones recientes de Windows 10.  WDAG recopila información de diagnóstico de mejora del producto sobre la apertura de sitios que no sean de confianza en un contenedor aislado, como cuánto se tarda en abrir una nueva ventana de Protección de aplicaciones.  

Con su permiso, WDAG también recopila información sobre cómo usa el explorador y la información sobre los sitios que visita.  Para desactivar la recopilación de datos de diagnóstico de Microsoft Edge en Windows 10, vaya a **Inicio** > **Configuración** > **Privacidad**y seleccione **Comentarios y diagnósticos**.  Para deshabilitar la recopilación de datos de diagnóstico para todas las demás plataformas, vaya a `edge://settings/privacy` en una sesión de exploración normal y desactive las opciones **Mejorar los productos de Microsoft enviando informes de bloqueo y datos sobre cómo usa el explorador** y **Mejore las búsquedas y los productos de Microsoft enviando información acerca de los sitios web que visita en Microsoft Edge**.  

## Windows Information Protection  

Windows Information Protection \(WIP\) ayuda a evitar la pérdida accidental de información corporativa.  Solo está disponible para las organizaciones mediante directivas de grupo que administre su organización.  WIP está habilitado para los sitios que se identifican como activos corporativos.  Identifique los sitios que son activos corporativos desde el icono de administración en la barra de direcciones.  WIP aplica características como la prevención de copiado y pegado desde el explorador o la carga de determinados archivos a sitios ajenos a la organización.  

:::image type="complex" source="./media/whitepaper-media/w-i-p.png" alt-text="Windows Information Protection" lightbox="./media/whitepaper-media/w-i-p.png":::
   Windows Information Protection  
:::image-end:::  

Si está activado el trabajo en curso de su versión de Microsoft Edge, el explorador recopila los registros de eventos y los envía a la organización.  Si WIP está activado, no podrá dejar de participar en la recopilación de datos.  WIP solo está disponible en las versiones de Windows 10 de agosto de 2016 o posteriores.  Para más información acerca de los registros de eventos capturados por WIP, consulte [Cómo recopilar registros de eventos de auditoría de Windows Information Protection (WIP)][WindowsSecurityInformationProtectionCollectAuditEventLogs].

## Ponerse en contacto con el equipo de Microsoft Edge  

El equipo de Microsoft Edge siempre está escuchando a los clientes y valora sus comentarios.  Para proporcionar comentarios en Microsoft Edge, seleccione **Configuración y más** > **Ayuda y comentarios**y seleccione **Enviar comentarios**.  Para Aplicaciones web progresivas \(PWA\), seleccione **Configuración y más (...)** y seleccione **Enviar comentarios a Microsoft**.  Debe proporcionar información detallada sobre los comentarios, pero el resto de la información es opcional.  Si se detecta un correo electrónico de su perfil de Microsoft Edge, se rellena previamente, junto con la URL actual del sitio en el que se encuentra y los datos de diagnóstico relevantes.  Los datos de diagnóstico pueden incluir datos sobre las características de Microsoft Edge activadas y sobre el uso del explorador.  También puede incluir una captura de pantalla, archivos de su dispositivo y la grabación de su explorador.  La captura de pantalla, los archivos o la grabación opcionales que proporcione pueden incluir información de identificación personal.  Los datos se usan solo para propósitos de diagnóstico y mejora del producto.  

Los comentarios de los usuarios se envían de forma segura a Microsoft mediante HTTPS y se almacenan en servidores seguros de Microsoft.  Si incluye su dirección de correo y la opción **Mejorar el uso de los productos de Microsoft enviando informes de bloqueo y datos sobre cómo usa el explorador** está activada en la configuración de privacidad de Microsoft Edge, un identificador único para la instalación de su explorador en su dispositivo se asocia a sus comentarios.  Todos los datos de diagnóstico, incluidos registros de diagnóstico, grabaciones y datos adjuntos, se almacenan durante hasta 30 días.  El resto de los datos de los comentarios, incluida una captura de pantalla opcional, se almacenan hasta 15 meses.  Haga una [solicitud][MicrosoftConcernPrivacy] para eliminar sus comentarios si ha proporcionado un correo electrónico con el elemento de comentarios.
